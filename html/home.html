What is Pejama with guys ?

Pejama with guys is a 2D platoformer game with wave spawning system the Goal of the game is collecting kills there is no levels because this is the beta version of the game and the first experience in game development .



What language is used in developing this game ?

GDScript evolved as a custom scripting language to be used with the Godot computer (and mobile device) Game engine.
GDScript is easy to learn with a syntax similar to Python where the code is indented rather than using lots of brackets and semi-colons. allows for rapid code development and leverages the rich library of available functions within the Engine for fast enough speed of execution for the Game Logic in most cases.
